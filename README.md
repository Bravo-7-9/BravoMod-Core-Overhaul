
# BravoMod Core Overhaul

An overhaul of various Vanilla Minecraft features to create a suitable base for my futures mods.

## Features:
### Arrow Rework
  * Arrowheads
    * Early Game
      * Spruce
      * Bone
      * Flint
      * Obsidian
    * Mid-Late Game
      * Bodkin Point
      * Fixed Blade Broadhead
  * Arrow Shafts
    * Bamboo
    * Wood
  * Arrow Fletchings
    * Bare
    * Feather
    * Copper Vanes
  * Gimmicks
    * Bodkin Points pierce armor
    * Broadheads inflict bleed
    * Bowfishing
